# My-Web

个人网站项目，用于展示个人信息与相关内容。

## 部署状态

[![Netlify Status](https://api.netlify.com/api/v1/badges/ed2b4bb9-0d5e-44dd-b82d-5238624b3160/deploy-status)](https://app.netlify.com/projects/strong-blini-998ae9/deploys)

## 项目介绍

本仓库是基于开源项目[MyWeb](https://github.com/Twilightsnow2009/MyWeb)修改而来的个人网站实现。

网站主要功能：
- 展示个人基本信息
- 呈现个人简介与相关内容
- 响应式设计，适配不同设备浏览

## 代码说明

- 仓库代码可能会随时进行改动和更新
- 核心页面文件为`index.html`，包含完整的页面结构与样式
- 页面设计简洁，包含个人头像、信息介绍和版权声明等元素
